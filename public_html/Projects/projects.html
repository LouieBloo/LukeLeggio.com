Eggbryo

Unfortunately Chrome, Firefox, and IE have been phasing out plugin based web applications. Because of this, the Unity Web Player will no longer work on a lot of browsers, so I won’t be able to show you some projects through this site; however, you can download them and run them locally.

Eggbryo is the latest game I’ve been working on in my free time. It is a quick paced game with a simple task; save the eggs before they freeze to death. The player must switch between the colored boxes on either side of the screen before they can click an egg. If you have the wrong colored box selected and you click an egg that egg will die. There are many kinds of eggs and even a few powerups to help the player.

Eggbryo was created entirely in Unity, and is 100% of my own code and design. The game is aimed at the Android platform and just released on the Play Store! Go check it out!

Download Notes:

To run the game just open the .exe file in the source folder. Since the game is targeted for mobile release it will ask you for your graphics settings, so make sure you select "Fantastic", have the windowed button enabled, and choose a resolution at or above 1280x720.

-Press Q to quit.

Mosaic Creator

Mosaic is an itty bitty program I made just for the fun of it. It will take in a source picture as well as a source folder, and recreate the source picture from all the images in the source folder. I accomplish this by scaling all of the source folder's pictures to the desired dimensions, iterating over each pixel on the source picture, finding the source folder picture that best matches that pixel color, and then creating a giant image from the results.

Download Notes:

To run the program just open the .exe in the folder. Just like Eggbryo make sure you run it in WINDOWED mode, and follow the directions on the screen.

Have Fun!

The Last Dawn & Base Defense

The Last Dawn and Base Defense were the first two games I ever created. These games were written entirely in C++ and use the SDL2 libraries to display the graphics. The engine for the games were written from scratch, and include features such as: dynamic A* pathfinding/movement, automated turrets, walls, particles, weapon pickups, item pickups, health systems, upgrade systems, reloading, minimaps, and collision detection.

I was writing the engine and making the games simultaneously; therefore, there wasn't a whole lot to do in either of them. I quickly moved on to developing within Unity as it took a lot of the overhead out of the equation. I put these games up mainly to display where I’ve come from, and to show that I have a deeper understanding of game engines aside from just using them.

Download Notes:

The only game you can download currently is The Last Dawn since I've lost the source code for Base Defense. Base Defense had my updated engine and the A* pathfinding. This is an EXTREME prototype! There is barely any functionality other than picking up weapons, shooting, and opening the backpack. To run the game open up the folder and run ZombieRPG.exe

-WASD to move
-1,2,3 to switch weapons
-Left click to fire
-R to reload
-T for backpack
-Spam Right click to quit
